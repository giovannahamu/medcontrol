# Changelog

Todas as mudanças notáveis neste projeto serão documentadas aqui.

## [1.0.0] - 2025-01-01

### Adicionado
- Cadastro de medicamentos com nome, dose e horários
- Listagem de todos os medicamentos cadastrados
- Remoção de medicamento por ID
- Busca por nome (parcial, sem case-sensitive)
- Alerta de medicamentos para o horário atual
- Persistência em arquivo JSON
- Testes automatizados com pytest
- Linting com ruff
- Pipeline de CI com GitHub Actions
